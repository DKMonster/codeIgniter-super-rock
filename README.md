# A Codeigniter API Manager #

## What is this repository for? ##

You can easily to manage own codeigniter api library.

## Table of Contents ##

* Installation
* Usage
* FAQ

## Installation ##
Download this library and put into the codeigniter project.

### config/config.js ###
We had import the libraries include:
#### libraries ####
* session
* getpost
#### helper ####
* url
* cookie (options)
#### config ####
* global
* routes_api
* code_error
* code_success
#### model ####
* mod_general
* mod_config
